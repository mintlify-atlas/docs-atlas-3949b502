---
title: "Institutions (Etablissements)"
description: "Manage the educational institutions your trainees come from"
---

An **établissement** is an educational institution — a university, school, or training centre — that a trainee comes from. Each trainee is linked to one institution, and the institution's name appears on generated attestation PDFs.

<Note>
  Register your institutions before adding trainees. Because each trainee must be linked to an institution, you should set up all relevant institutions first.
</Note>

## Listing institutions

Navigate to the **Etablissements** section from the sidebar. The page displays a table with the following columns:

| Column | Description |
|--------|-------------|
| Nom | The name of the institution |
| Adresse | The physical address of the institution |

Each row has edit and delete action buttons.

## Adding an institution

<Steps>
  <Step title="Open the create form">
    On the Etablissements list page, click the **+ Ajouter nouveau Etablissement** button.
  </Step>
  <Step title="Enter the institution name">
    Fill in the **Nom** field with the full name of the institution. This field is required.
  </Step>
  <Step title="Enter the address">
    Fill in the **Adresse** field with the institution's physical address. This field is required and will appear on attestation documents generated for trainees from this institution.
  </Step>
  <Step title="Save">
    Click **Ajouter** to create the institution. You will be redirected back to the institutions list. Click **Annuler** at any time to discard the form and return to the list.
  </Step>
</Steps>

## Editing an institution

1. On the Etablissements list page, click the edit icon next to the institution you want to update.
2. The edit form opens pre-filled with the institution's current **Nom** and **Adresse**.
3. Update either field as needed and click **Modifier** to save your changes.
4. Click **Annuler** to discard your changes and return to the list.

<Note>
  Updating an institution's name or address will be reflected on any attestation PDFs generated after the change. Previously generated PDFs are not affected.
</Note>

## Deleting an institution

On the Etablissements list page, click the delete icon next to the institution you want to remove. A confirmation dialog will appear. Click **OK** to proceed.

<Warning>
  An institution cannot be deleted if trainees are currently linked to it. You will see an error message if this is the case. Reassign or remove the associated trainees first.
</Warning>
